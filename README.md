# Fable Kickstart

Convierte a Claude en tu socio tecnico para iniciar cualquier proyecto — y terminarlo.

Cuestionario critico → abogado del diablo con veredicto → analisis de negocio → plan verificable → construccion con deploy desde el dia 1 → cierre de verdad. Con memoria persistente (`proyecto/`) para retomar cuando sea, y la regla anti-scope-creep: las ideas de medio camino van a `ideas.md`, no al alcance.

**Uso:** copia la carpeta, abre `claude`, escribe `/iniciar`.

7 skills de metodo incluidos (5 de la guia [Fragmento de Fable](https://www.tododeia.com/community/fragmento-de-fable) de tododeia + 2 propios: negocio y cierre).
